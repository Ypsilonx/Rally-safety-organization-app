# Code of Conduct

## Nas zavazek

Chceme udrzovat otevrene a bezpecne prostredi pro vsechny contributory bez ohledu na vek, zkusenosti, identitu nebo puvod.

## Ocekavane chovani

- Komunikuj s respektem a vecne.
- Kritizuj napady, ne lidi.
- Poskytuj konstruktivni feedback.
- Predpokladej dobry umysl ostatnich.

## Nepripustne chovani

- Osobni utoky, urazky, trolling.
- Obtezovani v soukromych i verejnych kanalech.
- Sdileni citlivych udaju bez souhlasu.

## Vynuceni pravidel

Maintainer muze nevhodne prispevky nebo komentare upravit, skryt nebo odstranit. Pri opakovanem porusovani muze omezit moznost dalsi spoluprace.

## Hlaseni problemu

Pokud narazis na poruseni pravidel, zaloz issue s nazvem `CoC: <kratky popis>` nebo kontaktuj maintainera primo pres GitHub profil.
